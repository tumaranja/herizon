# Herizon Talent Matcher routing gate mockup

A solution idea for the challenge in [this LinkedIn post](https://www.linkedin.com/posts/mariluukkainen_hire-diverse-pre-vetted-talent-share-7466407920717545474-Serd/): match.herizon.io is for companies hiring diverse, pre-vetted talent, but job seekers keep mistaking it for their funnel and creating company profiles.

[index.html](index.html) is a routing gate in front of the matcher: visitors pick "I'm hiring" or "I'm looking for work" and get sent to the right place.

Check it out: [herizon-iota.vercel.app](https://herizon-iota.vercel.app)
