# Herizon Talent Matcher routing gate mockup

A solution idea for the challenge in [this LinkedIn post by Mari Luukkainen](https://www.linkedin.com/posts/mariluukkainen_hire-diverse-pre-vetted-talent-share-7466407920717545474-Serd/): match.herizon.io is for companies hiring diverse, pre-vetted talent, but job seekers keep mistaking it for their funnel and creating company profiles.

[index.html](index.html) mocks up a full-viewport routing gate in front of the matcher: visitors pick "I'm hiring" (continue to the matcher) or "I'm looking for work" (routed to join.herizon.io) before seeing anything else.
